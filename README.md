# Alessandro — Portfolio

Personal portfolio built with Astro, Tailwind CSS v4, and GSAP.

## Stack

- [Astro](https://astro.build/) v6
- [Tailwind CSS](https://tailwindcss.com/) v4
- [GSAP](https://gsap.com/) — scroll-driven animations
- [Web3Forms](https://web3forms.com/) — contact form

## Environment variables

The contact form requires a [Web3Forms](https://web3forms.com) key set in the hosting platform:

```env
PUBLIC_WEB3FORMS_KEY=your_key_here
```
